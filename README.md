## How to build
 - Install [raylib](https://github.com/raysan5/raylib). Build instructions [here](https://github.com/raysan5/raylib#build-and-installation).
 - Install libvlc. On debian/ubuntu/etc.: ```sudo apt-get install libvlc-dev```
 - Run ```./build.sh```
